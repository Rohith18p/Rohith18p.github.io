SLAM with SPOT

This project involved large-scale indoor data collection using Boston Dynamics Spot, equipped with five stereo cameras, a Velodyne LiDAR, and Spot’s internal odometry. Data was recorded across multiple environments, with an external Jetson Orion Nano handling collection through Spot’s I/O interface. Remote operation was managed via SSH with custom ROS launch files. For mapping, RTAB-Map SLAM is implemented in ROS, generating detailed 3D point clouds from the collected datasets.
