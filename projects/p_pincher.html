RTABMap SLAM with SPOT

Introduction:

This project demonstrates an intelligent pick-and-place system using the PincherX100 5DOF robotic manipulator, integrating MATLAB simulations, trajectory planning, inverse kinematics, and obstacle avoidance techniques. The implementation progressed through three key stages, transitioning from simulation to real-world application, with the incorporation of advanced path-planning algorithms.

Stage 1: MATLAB Simulation

The first stage involved simulating the pick-and-place task using MATLAB. The object and goal positions were predefined, enabling controlled experimentation. Key steps included:

1. Trajectory Planning: Generating smooth waypoint paths to transition between positions.
2. Inverse Kinematics (IK): Using MATLAB’s IK solver to calculate the joint positions corresponding to each waypoint.
3. Simulation Execution: Feeding the calculated joint positions to a virtual PincherX100 model to successfully perform the pick-and-place task.

Stage 2: Real-World Implementation

The second stage translated the MATLAB simulation logic to the physical PincherX100 manipulator. The system was enhanced with a vision setup comprising an overhead camera and AprilTags to dynamically detect object, goal, and obstacle positions.

Trajectory Generation: A generalized cubic polynomial function was implemented to create smooth and abrupt-free paths for each phase Start-to-object location, object-to-goal, goal-to-home.

Execution: The calculated trajectories were fed to the PincherX100 manipulator, enabling precise pick-and-place actions.

Stage 3: Obstacle Avoidance

Geometric Method:

Marked the obstacle’s center and defined a large circle with a safety offset. Generated a trajectory comprising of A straight line from the current position to the circle’s boundary, A curve along the circle until exiting at the next intersection with the line. A straight line to the goal position. A straight line to the goal position.

A* Method:

Represented the workspace as a grid, marking obstacles as restricted zones. Used the A* algorithm to compute an optimal collision-free path from the current position to the goal.

Inverse Kinematics Approaches:

To enhance flexibility and optimize manipulator performance, three inverse kinematics (IK) methods were utilized:

1. Default Solver: Leveraged the built-in solver for quick implementation.
2. Analytical Solution: Derived a closed-form solution tailored to the 5DOF manipulator.
3. Gradient Descent: Implemented to compute optimal joint configurations for desired positions, ensuring smooth and precise movements.

Conclusion:

This project successfully integrated simulation, vision systems, trajectory planning, and obstacle avoidance, demonstrating the versatility of the PincherX100 manipulator. By employing advanced methodologies such as A* and geometric planning, the system achieved reliable and adaptive pick-and-place functionality. The combination of analytical solutions and optimization techniques further optimized the manipulator’s performance.
