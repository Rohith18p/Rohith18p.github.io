Non-Prehensile Manipulation

using MPPI

This project tackles non-prehensile manipulation with complex, high-DOF manipulators using a Model Predictive Path Integral (MPPI) controller. Instead of relying on hard-to-derive dynamics models, the approach integrates the PyBullet physics simulator directly into the MPPI loop, enabling real-time trajectory rollouts through sampling and importance weighting. This simulator-in-the-loop strategy eliminates the need for explicit modeling, making MPPI broadly applicable to highly articulated robots.
