Non Prehensile Manipulation using MPPI & Isaacgym

Details

This project is part of my Mobile Robotics course, explores non-prehensile manipulation using Model Predictive Path Integral (MPPI) control integrating with Isaac Gym. I aim to implement this approach across various robot platforms to evaluate its effectiveness in dynamic manipulation tasks. Details will be updated as the project progresses.

Research Paper: https://ieeexplore.ieee.org/document/10856359
