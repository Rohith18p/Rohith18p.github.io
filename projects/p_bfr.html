Ball Following Robot

Details

This project demonstrates integration across computer vision, IoT-based control, and robotics, showcasing real-time decision-making for autonomous tracking applications.

I developed a compact ball-following robot capable of autonomously tracking and approaching a target ball using vision-based control. The robot uses a mobile phone mounted on its chassis to capture real-time video, which is streamed online to an external system for processing.

The system employs contour detection algorithms to isolate the ball from the video feed by masking it within the frame. The centroid of the detected ball is calculated, and its position relative to the center of the screen determines the robot’s movement. If the ball is to the left of the center, the robot turns left and if the ball is to the right, the robot turns right. The robot continues to track and approach the ball until the detected ball’s size surpasses a predefined threshold, indicating proximity, at which point the robot stops. A NodeMCU controller manages motor operations, receiving commands from the external system over Wi-Fi.
